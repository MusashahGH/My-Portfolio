## ✨ Features

- 📱 **Responsive design** — works smoothly across desktop, tablet, and mobile
- 🎬 **Animated home section** — eye-catching entrance animations powered by Anime.js
- 🖼️ **Project showcase** — interactive swiper carousel to browse featured work
- 📄 **Downloadable resume** — recruiters can grab my CV in one click
- ✉️ **Contact section with email copy** — one-click email copy, no app switching needed
- 🌀 **Scroll-triggered animations** — sections reveal smoothly as you scroll
